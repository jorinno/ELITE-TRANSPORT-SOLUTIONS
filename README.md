# ELITE TRANSPORT SOLUTIONS
##Description
    #Functional 
      This subsystem helps users estimate transport costs 
      User enters distance to travelled and waiting time.
      User then selects type of transport(Boda Boda, Taxi, or Special Hire) as present of now.
      #concept of event driven programming
      User performs an event(button click in this case) 
                                                      which triggers the system to calculate and display the estimated fare
    #non functional
    System needs to keep record of satsified users and to enable that, intent users should signup(if don't have account), or       login(if already have an account).
    #New user signup
    user clicks signup button(an event) which triggers update of the UI, updated UI enables user enter the required                information, user clicks submit(event) which trigers display of a UI that confirms submission

    #setup
    subsystem prototype so far runs as a desktop application
